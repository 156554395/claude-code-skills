# 小说写作助手

一个帮助新手作者快速生成网络小说的 AI 技能。支持多种主流网文类型，自动生成大纲并逐章创作完整小说。

## 快速开始

### 基础用法

直接说「写小说」，技能会引导你完成：

1. 选择小说类型（玄幻、都市、科幻等）
2. 选择风格定位（热血、搞笑、悬疑等）
3. 设定预计字数
4. （可选）描述核心创意或主角设定

### 参数化用法

```bash
写小说，类型=玄幻，字数=10万，风格=热血
写小说，类型=都市，主题=重生复仇，字数=5万
```

## 支持的风格定位

- **热血爽文** - 快节奏、打脸、升级
- **轻松搞笑** - 幽默、轻松、日常
- **深沉黑暗** - 严肃、暗黑、人性
- **悬疑烧脑** - 推理、谜题、反转
- **情感治愈** - 温馨、感人、治愈

## 工作流程

```
需求收集 → 大纲生成 → 确认大纲 → 章节生成 → 完成
   ↓          ↓          ↓          ↓        ↓
 5个问题   AI自动生成   一次性确认   全自动    保存文件
```

### 1. 需求收集

AI 会问你以下问题：

- **必填**：小说类型、风格定位、预计字数
- **选填**：核心创意/主题、主角设定偏好

### 2. 大纲生成

AI 自动生成包含以下内容的完整大纲：

- **基础信息**：标题、类型、风格、字数、章节数
- **世界观设定**：等级体系、势力划分、背景设定
- **人物设定**：主角、配角、反派
- **剧情结构**：三幕式剧情规划
- **章节规划**：每章标题和核心剧情点

### 3. 确认大纲

你可以：

- 确认后继续生成
- 要求修改局部内容
- 要求重新生成

### 4. 章节生成

确认后自动逐章生成，每章约 3000 字，包含：

- 符合网文节奏的爽点设置
- 章末悬念钩子
- 连贯的剧情逻辑

### 5. 保存完成

小说保存为分章节的 Markdown 文件（每1章一个文件）：

```
./novels/小说标题/
├── 001章.md
├── 002章.md
├── 003章.md
└── ...
```

**优点：**

- 便于管理长篇小说
- 每个文件大小适中，方便阅读和编辑
- 可以按章节范围快速定位内容

## 常见问题

### 字数建议

- **5万字**：短篇小说，适合快速阅读
- **10万字**：标准网文长度，剧情完整
- **20万字以上**：长篇网文，剧情更丰富

### 如何让小说更好看？

1. **提供具体的主题创意**：如"重生复仇"、"系统流"、"废柴逆袭"
2. **设定主角特点**：性格鲜明、有独特背景或能力
3. **确认大纲时仔细检查**：这是调整剧情的最佳时机

### 支持断点续传吗？

如果生成过程中断（如字数过多），技能会自动保存已生成内容，下次可以从中断处继续。

### 小说质量如何？

技能基于经典网文套路生成，适合：

- 新手作者学习网文结构
- 快速生成故事框架进行修改
- 娱乐和创意启发

对于出版级质量，建议在 AI 生成基础上进行人工润色。

## 技巧提示

### 选择合适的类型组合

- 玄幻 + 热血爽文 = 经典升级流
- 都市 + 轻松搞笑 = 日常爽文
- 科幻 + 悬疑烧脑 = 硬核科幻

### 利用经典套路

如果你不确定写什么，可以让 AI 自动选择：

- 玄幻：废柴逆袭
- 都市：重生复仇
- 科幻：末世生存
- 历史：穿越改命

### 修改大纲

确认大纲时可以要求：

- 修改人物设定
- 调整剧情走向
- 增加或删除角色
- 改变结局

## 注意事项

1. **首次使用建议从5-10万字开始**，了解生成效果后再尝试更长篇幅
2. **大纲确认很重要**，这是调整剧情的唯一机会
3. **生成需要时间**，10万字小说大约需要15-30分钟
4. **生成后可手动编辑**，Markdown 格式方便修改

## 故障排除

| 问题       | 解决方法                                 |
| ---------- | ---------------------------------------- |
| 生成失败   | 检查 `./novels` 目录是否存在且有写入权限 |
| 内容不连贯 | 重新生成时告诉 AI 保持更严格的一致性     |
| 字数偏差   | 这是正常现象，AI 会尽量接近目标字数      |
| 大纲不满意 | 要求重新生成或具体指出需要修改的部分     |
